## 🔧 Firebase Configuration

The website is configured for Firebase but uses demo mode by default. To enable real Firebase:

1. Go to [Firebase Console](https://console.firebase.google.com)
2. Create project: `gamerdrift-auth`
3. Enable Authentication (Email/Password)
4. Create Realtime Database
5. Copy config and replace in `index.html`

See [API_INTEGRATION.md](API_INTEGRATION.md) for detailed setup.
